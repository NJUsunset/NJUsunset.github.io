秦时明月汉时关

摄影 / 谢杰

器材 / 尼康D750改机 + 腾龙35/1.4

参数 / f1.4，20s ，ISO1600，星空部分用赤道仪跟踪拍摄，多张拼接全景

地点 / 甘肃永泰古城

时间 / 2021年6月15日

黄河远上白云间，一片孤城万仞山。秦时明月汉时关，万里长征人未还……在苍凉的大西北，每当看到古老的城墙，脑海中很容易就浮现那些熟悉的诗词与意象，不自觉地把自己代入其中。似乎看到大汉天子的铁骑曾在这片土地驰骋，往来丝路的商队也曾在这里停歇，历史的车轮滚滚向前穿越数千年的风雨，朝代数经更迭，而头顶的星空永恒不变。有时候喜欢追问人生的意义，天地孤影任我行，世事苍茫成云烟，那些平时纠结的人或事，都在仰望星空的那一刻变得豁然开朗，明白何为自己内心所爱。看到这电影里似曾相识的城楼，很想像夕阳武士那样伫立城头对心中所爱大声地说出那三个字，星空为证！

神祗降临神山

摄影 / 蔡文毅

器材 / 尼康D750天文改机 + 腾龙35 1.4

参数 / f1.4，10s，ISO6400，7组拼接，每组10张堆栈

地点 / 西藏那曲萨普

时间 / 2021年10月5日

当萨普神山出现在面前时，你会发现之前为抵达它所爬过的坡，辗过的碎石路，淌过的冰川水，一切都是值得的。

风刮过白雪皑皑的山脊，星子倾巢而出挂满天空。入夜时分，银河中心准确无误地落在秘境山尖之上，如天梯，接引神祇；正有流星穿银河而过，长长的尾焰如利箭，划破高原雪域的夜空，恰如神祇降临。

象鼻星云

摄影 / 深蓝

器材 / 裕众102 + ZWO2600MM

参数 / SHO 888 单张600S

地点 / 香格里拉远程台

时间 / 2021年10月11日

象鼻管星云位于仙王座巨大的游离气体区域IC1396内，是一片距离地球约2,400光年的星际物质高度集中区域。这块星云被称为IC1396A，呈现出黑暗、高密度的球状。由于在可见光的波长下，这里是一个有着明亮蜿蜒边缘的黑暗斑块，俗称为象鼻管星云。

在绵延数百光年的发射星云IC1396之内，光亮的云气和黝黑的尘埃云交织在一起，而且恒星不停地从这个离我们只有三千光年远的星云诞生出来。这张色泽特别亮丽的星云影像，是由透过窄波段滤镜拍摄的数张数字影像合成出来的。 这些滤镜实际上挡掉了大部分星光，只有星云中某些特定种类原子所发出的光波才能穿透过去，这样,我们可以分辨出那哪些元素是IC1396辉光的主要来源。例如，以红色标示的是硫原子发出的辐射，氢的辐射是绿色，而氧是蓝色。这种美丽且具有科学价值的影像，和我们肉眼所见的图像是有很大差别的。

鹰星云与欧米伽星云广域

摄影 / 王楠

器材 / Takahashi FSQ 106/ ASI + ZWO ASI6200MM Pro

参数 / RGB77300S

地点 / 云南

时间 / 2021年7月

欧米伽星云距离我们约5500光年，坐落在射手座。在它的云气深处，一直有新的恒星诞生。鹰星云坐落在巨蛇座，距离我们约7000光年，同样也是个恒星诞生区。

海鸥星云（IC2177）

摄影 / 侯鹏

器材 / FSQ106 + ASI6200mmPro + azeq6

参数 / SII1200s30，Ha1200s30，OIII1200s30

地点 / 丽江

时间 / 2020年12月16日

这片色彩斑斓、线条优美的迷人星云被称为海鸥星云，因其与飞行中的海鸥相似而得名。这个区域由尘埃、氢、氦和较重元素的痕迹组成，是新恒星炽热而充满能量的诞生地。

14GM下的阿拉善大峡谷星轨

摄影 / 马青伟

器材 / 索尼a7r3 + 14GM

参数 / 15s，ISO3200，400张堆栈

地点 / 阿拉善大峡谷

阿拉善大峡谷的光污染还是有点重，银河的方向拍星的灯光闪烁不停。但我也不能辜负了这么好的夜晚，就面向光污染较少的方向拍摄延时，捕捉流星，顺便可以堆栈一组星轨。事实证明，星轨用延时的素材来堆栈还是差异很大，高感下噪点无法消除，画面里星星密集，形成了密集的星轨，不太素雅。但我依旧很着迷，着迷于拍摄的过程，以及天空的气辉，星轨的颜色……我想，经历了，总归是上帝赐予我的美丽风景。

绿汁七十二道拐星空

摄影 / 李剑云

器材 / D800 + 尼康AF-S + 尼克尔 14-24mm f 2.8G

参数 / f2.8，120s，ISO800，3张堆栈合成；天空：F2.8，120s，ISO800，2张拼接，赤道仪跟踪

地点 / 云南省玉溪市易门县

时间 / 2021年9月10日

云南多山，盘山公路是一大特色，这些险峻的公路两边无不伴随着美丽的风景，而夜晚升起的银河更是这蜿蜒的人工奇迹上空绝美的风景。曾经的“滇中铜都”，中国八大铜矿之一，伴随着铜矿资源的枯竭，铜都的历史也随之不断承载着激变。人走了，矿山沉寂了，绿汁72道拐，曾经被誉为天险的地方，弯弯曲曲的道路，当年车水马龙，日夜奔腾着运送矿产的汽车洪流，如今已很少有车走了，稀稀落落地走过几辆车，卷着一地的灰尘缓慢地奔向前方。
