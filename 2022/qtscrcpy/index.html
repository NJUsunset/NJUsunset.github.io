QtScrcpy——连接你的安卓设备

最近闲来无事在b站做了实名认证，遂试验了啊b的直播功能。由于平日主要游戏为手游，故寻找搭建了手机游戏投屏至pc推流的平台。寻觅中在某同性交友网站(github)偶得一方便的投屏软件QtScrcpy。今分享于此，望于诸君有助。

软件的要求和获取

要使用QtScrcpy链接安卓设备至pc，需要安卓设备的安卓版本大于等于5.0。软件提供了mac、win32和win64的编译包，其余平台可前去项目地址从源码编译。

软件本身开源，遵循Apache License, Version 2.0协议。

项目地址在：barry-ran/QtScrcpy: Android real-time display control software (github.com)

考虑到网站可能需要魔法，留一度盘链接（v2.0.1）：https://pan.baidu.com/s/1istzmHzATWw3ADZ0uqYZgw?pwd=hjqa 提取码：hjqa

软件的简单使用

下载软件包之后解压，然后双击QtScrcpy.exe运行程序（以win64为例），软件主界面如下：

QtScrcpy支持有线和无线两种连接方式，个人常用的为有线连接。安卓手机端需要前往开发者选项中开启USB调试模式。然后只需将目标安卓手机通过相应数据线连入电脑的usb接口（实测usb2.0也可，毕竟我手机端的typeC也是usb2.0速率的，两侧都是更高速的接口当然更好），之后在pc端点击软件界面左上侧的USB Connect按钮。一切正常的话，你会在电脑屏幕上看到如下一幕：

此时你可以把鼠标当作手指自由地控制你的安卓设备。软件会自动识别横屏状态并且切换显示如下。

如果想连接设备音频，QtScrcpy也集成了著名的sndcpy软件。首次使用时需要点击右侧UBS line部分中的install sndcpy按钮在手机端安装软件sndcpy。待安装完成，在pc端点击前一个按钮右侧的start audio按钮，待手机上显示出通知sndcpy:Audio forwarding enabled，便可以在pc端听到手机上应用输出的声音了。

一些小问题

QtScrcpy除开投屏，还可以在投屏时候同时录屏，在右上侧config中简单设置即可。

个人使用过程中遇到一些问题，记录如下：

  • 某些游戏不能正常保持低延迟投屏（自用测试实况足球国际版有此问题）

  • 某些游戏不能正常输出声音（自用崩坏三有此问题）

其余包括王者荣耀、b站客户端、懂球帝等一众软件使用正常，延迟控制符合介绍的延迟3070ms，fps3060。

总结

QtScrcpy为一小巧轻便的安卓投屏软件，其最大优势为免费和安卓端无需安装复杂的软件和操作，在简单的直播等投屏场景很好用。
